Date: 11/13/2023
Welcome to my website, I'm Howisthisaname. I might change usernames soon.

On this website, I plan to post random thoughts, post about my own creations, and general stuff like that. Maybe you care because you like taking a look into some other person's life. Maybe you don't care at all. Either is fine. Thank you for being here regardless.

Currently, the site is WIP, but starting in January 2024 I plan to have it finished in terms of layout.

The theming of the website is provided by deathau. It's a theme for Obsidian, a markdown text editor that I recently discovered (at the time of writing this) and think is cool. The files of this site exist as markdown files on my hard-drive, but are converted into HTML via a plugin for Obsidian created by KosmosisDesire, which also converts the theming mentioned above into HTML.
